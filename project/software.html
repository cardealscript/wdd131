Bachelor's Degree in Software Development

BYU-Idaho offers a fully online Software Development degree through BYU-Pathway Worldwide. Affordable, accredited, and designed for working students. No prior programming experience required.

Start PathwayConnect → View Degree Details

Learning Roadmap

1

PathwayConnect — Foundation

A 6-month programme that builds university-level skills in writing, mathematics, and personal, professional, and academic development. You also begin your degree immediately — the first course of the Software Development programme, Introduction to Python, is part of PathwayConnect.

2

Web & Computer Programming

Continue with Python and C#, then move into the three core front-end technologies — HTML, CSS, and JavaScript. This track builds a strong foundation as a front-end developer and prepares you for the Associate degree.

3

Build & Deploy Projects

Use GitHub to host and version control your code. Deploy to GitHub Pages. Build a real portfolio that employers can see — this is where your skills become visible to the world.

4

Associate → Full Bachelor's Degree

The Associate degree already makes you a strong full-stack developer. Continue to the Bachelor's in Software Development from BYU-Idaho — completed entirely online over approximately 3 years from PathwayConnect.
