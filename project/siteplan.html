01

Site Name

EN

EduNexus

edunexus.io suggested domain

EduNexus combines "Edu" (Education) and "Nexus" (a connection point or hub). The name communicates exactly what the site does: it serves as a central connection point for online learners — linking institutions, tools, and resources in one organized space. It is modern, easy to remember, and immediately signals an educational purpose.

02

Site Purpose

EduNexus is a personal dashboard for online students, providing a curated directory of educational institutions (with direct links to student portals), essential digital tools for productivity and study, curated learning resources such as books and references, and a dedicated section on software development — including a spotlight on BYU-Pathway Worldwide and its Software Development programme. The site also uses JavaScript to offer basic content translation, making it accessible to non-English speakers. The layout follows a Dashboard with Sidebar + Main Content structure for fast, intuitive navigation.

05

Typography

Display / Headings

Syne

Used for page titles, section headings (h1–h3), logo text, and navigation labels. Bold geometric personality that conveys confidence and modernity.

Body / Reading

Instrument Serif

Used for descriptive paragraphs, resource descriptions, and article-style content. A refined serif that brings warmth and readability to longer passages.

Code / Mono / Labels

DM Mono

Used for code snippets, domain names, technical labels, version numbers, and badges. Crisp and clean monospace for technical credibility.
